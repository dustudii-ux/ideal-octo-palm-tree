DU Preference Sheet Builder 2026

DU Preference Sheet Builder 2026 is a free web application designed for Delhi University aspirants. It helps students generate a well-structured preference sheet for the CSAS admission process based on their selected course and preferences.

Features
Generate DU preference sheets instantly
Course-specific preference recommendations
Clean and mobile-friendly interface
Print and save as PDF
Completely free and open source
Tech Stack
HTML
CSS
JavaScript
GitHub Pages (Hosting)
Goal

To simplify the DU admission process and help students make informed preference-ordering decisions during CSAS counselling.

One-line tagline for the website

Generate your DU CSAS preference sheet in seconds — free, simple, and student-friendly.
