# ideal-octo-palm-tree
DU Preference Sheet Builder 2026 is a free, student-focused web app that helps Delhi University applicants create and organize their CSAS preference lists quickly and efficiently.

DU Preference Sheet Builder 2026 is a free web application designed for Delhi University aspirants. It helps students generate a well-structured preference sheet for the CSAS admission process based on their selected course and preferences.

To simplify the DU admission process and help students make informed preference-ordering decisions during CSAS counselling.
